This is a Meshcoms portal to unify all your Lora Meshcommunications from Meshcore and Meshtastic

RUNNING COMMANDS

npm install
python launcher.py


Open your Webbrowser and goto http://127.0.0.1:53657/index.html
Make a connection to your nodes by serial or WIFI

For example
Meshcore   <TCP-IP>:5000
Meshtastic <TCP-IP>:4403
